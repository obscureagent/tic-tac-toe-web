# Tic Tac Toe Web Game

A beautiful, fully-playable tic-tac-toe game built with vanilla HTML, CSS, and JavaScript.

## Features

- **Two-Player Mode**: Play against another human on the same device
- **AI Opponent**: Challenge an AI that plays smart moves (tries to win, blocks your wins, prioritizes the center)
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Smooth Animations**: Polished UI with hover effects and transitions
- **Easy Reset**: Start a new game anytime with the reset button

## How to Play

1. **Two Player Mode** (Default):
   - Player X goes first
   - Players take turns clicking cells
   - First to get 3 in a row (horizontally, vertically, or diagonally) wins!

2. **AI Mode**:
   - Click "Play vs AI" to enable AI opponent
   - You are always Player X (you go first)
   - AI plays as Player O
   - The AI uses a smart strategy: tries to win, blocks your wins, and prioritizes the center

## Files

- `index.html` - Game structure and board
- `style.css` - Beautiful gradient background, card layout, and responsive design
- `script.js` - Complete game logic with win detection and AI opponent
